# To run an Elixir program

```
elixir lib/servy.ex
```

## Run using `iex`
### Run iex, then compile and run
```
iex
c "lib/servy.ex"
```

### Run iex, compile, and run in one step
```
iex lib/servy.ex
```

### Use mix context to run application
```
iex -S mix
```

# To recompile/reload a module
If in an `iex` session:
```
r <name of module>
```
